# Проект "Библиотека": Демонстрация принципов ООП на PHP

Данный проект является учебным пособием и демонстрацией ключевых концепций ООП на языке PHP. Он моделирует простую систему управления библиотекой, где можно регистрировать издания (книги, журналы), пользователей и управлять процессом выдачи и возврата книг.

## Ключевые концепции:

*   **Абстрактные классы** для создания общих шаблонов.
*   **Интерфейсы** для определения "контрактов" поведения.
*   **Трейты** для переиспользования кода в разных классах.
*   **Пространства имён (Namespaces)** для логической группировки кода.
*   **Автозагрузка классов (PSR-4)** для отказа от `require` и `include`.
*   **Инкапсуляция** и разделение ответственности между классами.

## Структура проекта

Проект организован по принципам близким к современным PHP-фреймворкам:

```
├── src/                      # Основная папка с исходным кодом
│   ├── Core/                 # Ключевые классы
│   │   ├── Library.php       # Главный класс
│   │   ├── Member.php        # Класс пользователя библиотеки
│   │   └── LibraryItem.php   # Абстрактный класс для всех изданий
│   │
│   ├── Items/                # Конкретные типы изданий
│   │   ├── Book.php          # Класс для книг
│   │   └── Magazine.php      # Класс для журналов
│   │
│   ├── Contracts/            # Интерфейсы
│   │   └── Borrowable.php    # Интерфейс для того, что можно взять "на дом"
│   │
│   └── Traits/               # Трейты
│       └── HasUniqueID.php   # Трейт для добавления уникального ID
│
├── autoload.php              # Файл с логикой автозагрузки классов
└── index.php                 # Точка входа
```

## Как запустить

1.  Убедитесь, что у вас установлен PHP (версия 7.4 или выше).
2.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Maxim-Belyi/OOP-practice.git
    cd oop
    ```
3.  Выполните команду:
    ```sh
    php index.php
    ```
4.  Скрипт `index.php` выполнит тестовый сценарий: создаст библиотеку, добавит в нее книги, журналы и пользователей, а затем сымитирует процесс выдачи и возврата, выводя все шаги на экран.
